# Offline Chatbot RAG Frontend

Vue frontend for the multi-modal RAG backend.

## Setup

```bash
npm install
npm run dev
```

## Backend URL

Set `VITE_BACKEND_URL` to your backend base URL.

- Local example: `http://127.0.0.1:8000`
- Hugging Face backend: `https://shivareddy-03-offline-chatbot-rag-backend.hf.space`
